Date of birth

The pattern for collecting date of birth from users is to have 3 text input for Day, Month and Year, each with there own labels. These are found under 'Date of bith' text and a hint providing an example.

Accessibility success criteria

  • The 'Date of birth' text needs to be associated with all the fields that text fields
  • The hint text needs to be associated with the text inputs so they are announced as a screen reader users tab through the form
  • The 'Day', 'Month' and 'Year' labels need to be associated with the the relevant text fields
  • The year of birth needs to be a range that is not unusable for users using a spon wheel on a mobile device
  • The text needs to meet colour contrast requirements
  • The form needs to be usable when the font size is increased.
